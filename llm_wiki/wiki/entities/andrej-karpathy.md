# Andrej Karpathy

## Overview
AI researcher and engineer. Former senior director of AI at Tesla (Autopilot), co-founder of [[openai]]. Known for educational content (Zero to Hero, CS231n) and for popularizing practical AI workflows including the [[llm-wiki-pattern]].

## Key Contributions
- **CS231n** — Stanford's CNN course
- **Tesla Autopilot** — Led AI/computer vision team
- **Zero to Hero** — Neural networks from scratch (YouTube)
- **[[llm-wiki-pattern]]** — Agent-built personal knowledge bases (April 2026)
- **Auto-research loop** — LLMs improving their own skills

## LLM Wiki Pattern
In April 2026, described a shift in LLM usage: spending tokens on "knowledge manipulation" not code. His pattern uses an LLM agent to build and maintain a personal wiki. One research topic grew to ~100 pages, ~400K words — without him writing a single word. ^[raw/articles/karpathy-llm-wiki-tweet.md]

## Views on AI Agents
From Lex Fridman (2026): ^[raw/transcripts/karpathy-lex-fridman.md]
- Uses both managed and self-hosted agents — Claude Code for coding, Hermes Agent for automation
- Believes persistent memory is the key differentiator — "the agent that remembers you will always beat the one that doesn't"
- Advocates a portfolio approach — different agents for different tasks
