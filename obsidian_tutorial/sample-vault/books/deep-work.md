# Deep Work

> **Author**: Cal Newport | **Genre**: Productivity | **Rating**: ⭐⭐⭐⭐
> **Status**: Completed | **Finished**: April 19, 2026
> [[Atomic Habits]]

## Key Ideas

### What Is Deep Work?

Professional activities performed in a state of distraction-free concentration that push cognitive capabilities to their limit. These efforts create new value, improve skill, and are hard to replicate.

### Deep Work vs Shallow Work

| Deep Work | Shallow Work |
|---|---|
| Requires focus | Can be done distracted |
| Creates value | Low value, logistical |
| Rare and valuable | Common, easily replicated |
| Example: writing a spec | Example: replying to email |

### Four Philosophies

1. **Monastic**: Eliminate all shallow work completely
2. **Bimodal**: Dedicate days/weeks to deep work, rest to shallow
3. **Rhythmic**: Do deep work at the same time every day (habit)
4. **Journalistic**: Fit deep work in whenever you can (advanced)

### Rituals for Deep Work

- Define where and when you'll work
- Set rules (no internet, no phone)
- Support the work (coffee, walking, etc.)

## Quotes
